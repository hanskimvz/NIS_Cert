# D 타입 카메라 보안기능 시험준비
 

### 시작하기전에 미리 정해야 하는 조건 들
- 한국 정부기관(국정원)의 인증인 만큼, 제품에서의 Dahua라는 색깔은 모두 지우고 SEJIN tech의 브랜드로 신청
- Dahua 제품중 Bullet camera인  DH-IPC-HFW5449T-ASE-LED
- webpage 자체 제작(http://124.61.244.239:5173) - 현재 개발중(ID: admin, PW: vivasejin12**)
- 웹페이지 개발을 위해서는 api full 명세서가 필요함.
- 웹페이지의 영상을 websocket을 사용하는 것 같던데, 인증 방식등에 대해 설명한 내용이 필요함.

| 10. 환경·물리 | 포트봉인, 설정백업보호 | ⑦ 운용·연동구조 | 물리포트 보호정책, 설정 암호화 | 백업파일 암호화 확인 |

## 3. 엔지니어 실무 준비 체크리스트 & 제출 패키지 구조
### 3-1. 엔지니어 즉시 실행 체크리스트 (시험대비 현장용)
<u>엔지니어가 시험 직전에 실제 장비에서 반드시 확인해야 하는 항목을 체크리스트 형태로 정리.</u>

#### [계정·인증]
- 장비 초기화 후 기본계정 강제 변경 흐름 캡처 (로그 포함)
- PW 정책(최소 길이/복잡도/변경주기) 설정 화면 캡처
- 연속 5회 로그인 실패 시 계정 잠금 또는 지연 발생 로그 캡처
- 세션 타임아웃(미사용 10분) 동작 영상 또는 캡처

#### [네트워크·통신]
- RTSP Digest 인증 패킷 덤프 (요청/응답 흐름 확인 가능하게)
- TLS 1.2 Handshake 패킷 덤프(클라이언트–서버 간 협상 과정)
- 불필요 포트 비활성화 전/후 nmap 결과 비교 리포트

#### [저장·데이터 보안]
- 저장 암호화 설정 화면 캡처
- 테스트 파일(예: 더미 녹화본) 암호화 여부 확인(평문 접근 불가 확인)

#### [시스템 무결성]
- FW 업데이트 시 전자서명 검증 성공 로그 1건
- FW 업데이트 시 전자서명 검증 실패 로그 1건
- 부팅 무결성 검사 결과 로그 (Secure Boot 적용 시 화면 포함)

#### [로그·감사]
- 감사로그 로테이션 정책(용량/기간) 설정 화면
- 로그 무결성(해시/서명) 검증 예시 화면 또는 보고서

#### [연동·Syslog]
- Syslog(TLS) 전송 테스트 결과(조건부 항목)

#### [물리보안]
- USB/UART/JTAG 디버그 인터페이스 봉인 사진
- 관련 메뉴(콘솔 접근, 디버그모드 등) 비활성화 화면 캡처
